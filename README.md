API RESTful em .NET 8.0 e Vue.js

Para rodar o projeto abra 2 terminais, "dotnet run" na API e "npm run dev" no front ("AppCadastro_vue").

Os testes unitários foram desenvolvidos por meio do Framework "XUnit" e estão presentes na pasta "AppCadastro.Testes".

AppCadastro/ -> Projeto da API (.NET 8.0, Entity Framework InMemory)
AppCadastro_vue/ -> Front-end (Vue.js, Vite)
AppCadastro.Tests/ -> Testes unitários (xUnit)

Pré-requisitos:

- .NET 8 SDK;
- ⁠Node.js 18+;
- ⁠NPM.
